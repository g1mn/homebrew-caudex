# homebrew-caudex

Homebrew Cask for [Caudex](https://caudex.sh) - Modern GUI for Claude Code sessions.

## Installation

```bash
brew tap g1mn/caudex
brew install caudex
```

## Usage

After installation, launch Caudex from Applications or run:

```bash
open -a Caudex
```

## About Caudex

Caudex is a powerful desktop application that enhances your Claude Code experience with:

- **Session Management**: Resume past sessions instantly
- **Git Integration**: View branch, status, and commit history at a glance
- **MCP Server Management**: Connect and configure MCP servers with ease
- **Keyboard Shortcuts**: Efficient navigation with customizable hotkeys
- **Shell Terminal**: Run regular shell commands alongside Claude Code
- **Notifications**: Get notified when Claude Code completes or needs input
- **Remote Access**: Access your Claude Code sessions from any device

## Updating

```bash
brew upgrade caudex
```

## Removing

```bash
brew uninstall caudex
```

To remove all associated files:

```bash
brew uninstall --zap caudex
```
